Blog API Doc
Login
POST

"https://tourday.team/api/auth/login/"

Send either username or email address with password. If the credentials are valid then the response will contain a key. use that key to acces all data.

{
"username": "asif",
"email": "asif.qubit@mail.com",
"password": "******"
}
Signup
POST

"https://tourday.team/api/auth/signup/"

If the credentials are valid then the response will contain a key. use that key to acces all data.

{
"username": "asif",
"email": "asif.qubit@mail.com",
"password": "******"
}
Get Profile Info
GET

"https://tourday.team/api/profile/"

Token is required. Add token with request header.

Request:
{
"Authorization": "Token 93bc86220b144548e5bb507851b6ef7c2a5e1a14",
}
Response:
{
"username": "asif",
"profile": {
"name": "Asif Shahriar",
"email": "asif.qubit@gmail.com",
"fb": "asif.8617",
"insta": "_asif_shahriar",
"city": "Rangpur, Bangladesh",
"bio": "Full Stack web developer"
"picture": "/media/user/pic.jpg"
},
"is_completed": true
}
Update Profile Info
POST

"https://tourday.team/api/profile/"

Token is required. Add token with request header. Send only specific filed to update it.
Ex: If you want to update only name filed, just send name in the request. Keep other field blank.

{
"name": "Asif Shahriar",
"email": "asif.qubit@gmail.com",
"fb": "asif.8617",
"insta": "_asif_shahriar",
"city": "Rangpur, Bangladesh",
"bio": "Full Stack web developer",
"picture": picture.jpg
}
Get User Posts
GET

"https://tourday.team/api/get_posts/username"

Send username with the url and get the response containing all posts. If that user has more than 10 posts, you will get next page url in "next" in the response.

{
"count": "post count",
"next": "next page url",
"previous": "previous page url",
"results":
{
"id": 1,
"post": "post here"
"image": "/media/user/pic.jpg",
"date": "2020-09-16",
"location": "Dhaka",
"user": 1,
"liikes": [1,2,3]
}
}
Write Post
POST

"https://tourday.team/api/post/"

Use Location number instead of full location name. See location key & Value

{
"post": "post text here",
"location": 1,
"date": "yyyy-mm-dd",
"image": image.jpg
}
Delete Post
POST

"https://tourday.team/api/post_delete/"

{
"id": post_id,
}
