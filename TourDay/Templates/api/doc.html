API Documentation

Authentication
Login
POST

"https://www.tourday.team/api/auth/login/"

Send either username or email address with password. If the credentials are valid then the response will contain a key. use that key to acces all data.

{
"username": "asif",
"email": "asif.qubit@mail.com",
"password": "******"
}
Sign up
POST

"https://www.tourday.team/api/auth/signup/"

If the credentials are valid then the response will contain a key. use that key to acces all data.

{
"username": "asif",
"email": "asif.qubit@mail.com",
"password": "******"
}
Forget Password
POST

"https://www.tourday.team/api/forget_password/"

Use username or email address in username_email field. Use slug from response in reset_password url.

{
"username_email": asif.qubit@gmail.com
}
Response:
{
"status": 200,
"slug": "asif"
}
Reset Password
POST

"https://www.tourday.team/api/reset_password/username/"

Use code that sent to your email account. Use slug from forget_password response in reset_password url instead of username. Password must be greater than 7 characters.

{
"code": TKPHHKGC,
"password1": ********,
"password2": ********
}
Delete Account
POST

"https://www.tourday.team/api/delete_account/"

Send a POST request with user token the account will be deleted.

Response:
{
"status": 200
}
