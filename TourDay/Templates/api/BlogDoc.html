Add Post
POST

"https://www.tourday.team/api/blog/addpost/"

Token is required. Add token with request header.

Request:
{
"Authorization": "Token 93bc86220b144548e5bb507851b6ef7c2a5e1a14",
}
"title": "Blog Post title Here",
"description": "description here",
"image": "attach your BLOG image",
"division": "your division here, Ex. Rajshahi",
}
Update Post
PUT

"https://www.tourday.team/api/blog/edit/post_id"

Token is required, Request Token who create a post. Add token with request header. Send all required field to update it.

Request:
{
"Authorization": "Token 93bc86220b144548e5bb507851b6ef7c2a5e1a14",
}
{
"title": "Blog Post title Here",
"description": "description here",
"image": "attach your BLOG image",
"division": "your division here, Ex. Rajshahi",
}
Delete Post
DELETE

"https://www.tourday.team/api/blog/delete/post_id"

Token is required, Request Token who create a post. Add token with request header.

Request:
{
"Authorization": "Token 93bc86220b144548e5bb507851b6ef7c2a5e1a14",
}
Response:
"Delete Successfully."
